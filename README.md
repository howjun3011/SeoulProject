# Seoul-related public data provision service

### 공공데이터 API 및 json 데이터를 활용해 서울 지역 기반 실시간 데이터 서비스를 제공하는 웹 개발 프로젝트
<br>


개발 기간 : 2024.11.25 ~ 2024.12.19

담당 업무 : 프로젝트 기획 및 체육 탭 구현

프레임 워크 : Spring Boot, React, MyBatis

데이터베이스 : Maria DB

<br>
